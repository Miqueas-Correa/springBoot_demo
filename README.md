# RUTAS:

- Cliente:
  * /cliente
  * /cliente/{id}
  * Cuenta con: GET, POST, PUT y DELETE.

- Cuenta:
  * /cuenta
  * /cuenta/{id}
  * /cuenta/transferir/{id}
  * /cuenta/depositar
  * /cuenta/retirar
  * Cuenta con: GET, POST, PUT y DELETE.

- Movimientos:
  * /cuenta/movimientos
  * /cuenta/movimientos/{id}
  * Cuenta con: GET.
